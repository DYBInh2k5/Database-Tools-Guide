# README - Cloud Databases

## AWS

- Amazon RDS: Managed SQL cho PostgreSQL, MySQL, MariaDB, SQL Server, Oracle.
- Amazon Aurora: Hiệu năng cao, tương thích MySQL/PostgreSQL.
- DynamoDB: Managed key-value/document, scale lớn, phù hợp serverless.

## Google Cloud

- Cloud SQL: Managed SQL cho PostgreSQL, MySQL, SQL Server.
- AlloyDB: PostgreSQL managed cấp enterprise.
- Firestore: NoSQL document-native cho ứng dụng realtime.
- Firebase: Nền tảng BaaS đầy đủ cho mobile/web, tích hợp Firestore/Auth/Functions.

## Microsoft Azure

- Azure SQL Database: SQL managed cho hệ sinh thái Microsoft.
- Azure Database for PostgreSQL: PostgreSQL managed trên Azure.
- Azure Cosmos DB: Multi-model NoSQL, phân tán toàn cầu.

## BaaS / Startup-friendly

- Supabase: PostgreSQL managed + Auth + Storage + Realtime.
- Firebase: Firestore/Realtime DB + Authentication + Cloud Functions.
- Neon: Serverless PostgreSQL, hỗ trợ branch database cho workflow dev.
- PlanetScale: Serverless MySQL với quy trình schema branch/deploy.
- Aiven: Managed open-source data stack cho PostgreSQL, Redis, Kafka, OpenSearch.

## Khi nào nên chọn managed DB?

- Team nhỏ, cần ra sản phẩm nhanh.
- Muốn giảm chi phí vận hành hạ tầng DB.
- Cần HA/backup/monitoring có sẵn từ nhà cung cấp cloud.
