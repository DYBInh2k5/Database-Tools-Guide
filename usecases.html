Chọn Database theo bài toán cụ thể

Danh sách gợi ý nhanh để ra quyết định kiến trúc ban đầu.

Ecommerce / ERP / CRM

Gợi ý: PostgreSQL hoặc MySQL, thêm Redis cho cache.

Realtime feed / profile động

Gợi ý: MongoDB hoặc Couchbase; thêm OpenSearch nếu cần search.

Analytics logs

Gợi ý: OpenSearch cho tìm kiếm; InfluxDB cho metrics/time-series.

Anti-fraud / Recommendation

Gợi ý: Neo4j cho quan hệ đồ thị, kết hợp SQL cho nghiệp vụ lõi.

Scale đa region

Gợi ý: CockroachDB, TiDB, Cassandra hoặc ScyllaDB.

Serverless trên AWS

Gợi ý: DynamoDB hoặc Aurora Serverless tùy kiểu truy vấn.

Luôn benchmark bằng dữ liệu thật trước khi chốt hạ tầng.
